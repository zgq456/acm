# 问题描述

浙江工商大学终于也要有自己的JudgeOnline了！
但浏览器是不能直接显示一些特殊字符的。为了把选手的代码显示给大家看，需要把一些特殊的符号换成HTML标签。
比如:
1.'<'需要换成 "&lt;"
2.'>'需要换成 "&gt;"
3.' '(空格)需要换成 "&nbsp;"
4.'	'(TAB符)需要换成 "&#9;"
5.其他字符保持不变。
现在请你来完成这项工作。

# 输入

输入一段代码(可能有多行)，把里面的符号按上面的规则替换掉。

# 输出

输出替换好的代码。

# 输入示例

#include <stdio.h>

int main(void)
{
	return 0;
}

# 输出示例

#include&nbsp;&lt;stdio.h&gt;

int&nbsp;main(void)
{
&#9;return&nbsp;0;
}
